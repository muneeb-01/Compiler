SimpleLang Syntax Analyzer Project Summary

Overview:
This project implements a Syntax Analyzer (Parser) for SimpleLang, a simplified instructional programming language using Roman Urdu keywords. The parser is implemented using the Recursive Descent parsing technique in Python.

Key Features:
- Support for variable declarations (number, sach_ya_jhoot, jumla).
- Support for function declarations and calls.
- Control structures: agar (if), har (for), jabtak (while).
- Arithmetic and relational expression parsing.
- Detailed error reporting with line numbers and descriptive messages.

Keywords Mapping:
- int    -> number
- bool   -> sach_ya_jhoot
- string -> jumla
- void   -> kuch_nahi
- if     -> agar
- for    -> har
- while  -> jabtak
- return -> wapis
- true   -> sach
- false  -> jhoot

Files Included:
1. grammar.txt: Formal EBNF specification.
2. lexer.py: Lexical analyzer (tokenizer).
3. parser.py: Recursive descent syntax analyzer.
4. main.py: Execution script for running tests.
5. test1_valid.txt: Valid program using all constructs.
6. test2_error_decl.txt: Program with syntax error in variable declaration.
7. test3_error_brace.txt: Program missing a closing brace.
8. test4_error_assign.txt: Program with invalid assignment expression.
9. test5_error_func.txt: Program with incorrect function declaration.

Testing Results:

Instructions:
To run the parser on all tests, execute:
python main.py

To run on a specific file:
python main.py <filename.txt>
